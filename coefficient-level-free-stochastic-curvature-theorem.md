# Coefficient-Level Free Stochastic-Curvature Theorem

Author: Felix Robles Elvira

Date: 2026-05-12

Status: theorem-level proof at the leading coefficient scope. This proves the coefficient-level stochastic-curvature theorem for the free one-particle `1+1D` Dirac ISP, using the exact collar-excision leading coefficient and the exact bond-centered thin-slab theorem already established in the free-core papers. It also proves the normalized `lambda`-family version at the same leading scope.

Follow-up: `finite-slab-free-stochastic-curvature-theorem.md` proves a narrow Tier 2 finite-slab lift for log-smeared exact comparison maps under the coupled scaling `Delta(a)/a^2 -> 0`. Broader primitive-kernel smooth-lapse finite-slab regularizations still require separate proofs.

## 1. The Result

Let `N,M` be smooth compactly supported real lapse profiles on the line, and let `phi` be a smooth compactly supported two-component spinor. Let

```math
\beta=N\partial_xM-M\partial_xN.
```

Let `iota_a` be the lattice sampling map

```math
(\iota_a\phi)_{n,s}=a^{1/2}\phi_s(an),
```

on a large periodic lattice whose seam is outside the supports of `N`, `M`, and `phi` for all sufficiently small `a`.

Let `A_n^{(1)}` be the singleton collar-excision coefficient in

```math
J_n(\Delta)=I+\Delta^2A_n^{(1)}+O(\Delta^4).
```

Define the leading coefficient-level smeared deformation operator

```math
\mathcal K_a[N]
:=
a\sum_n N(an)A_n^{(1)}.
```

Define the coefficient-level stochastic curvature by

```math
\mathfrak C_a[N,M]
:=
[\mathcal K_a[N],\mathcal K_a[M]].
```

Then the coefficient-level free stochastic-curvature theorem is:

```math
\mathfrak C_a[N,M]\iota_a\phi
=
\iota_a\left(\left(\beta\partial_x+\frac12\partial_x\beta\right)(I-\alpha)\phi\right)
+O(a)
```

in the lattice `ell^2` norm. Equivalently,

```math
\lim_{a\to0}
\left\|
\mathfrak C_a[N,M]\iota_a\phi
-
\iota_a K_\parallel[\beta]\phi
\right\|_{\ell^2}=0,
```

where

```math
K_\parallel[\beta]
=
\left(\beta\partial_x+\frac12\partial_x\beta\right)(I-\alpha).
```

For the admissible `lambda` family

```math
U_n^{(\lambda)}(\Delta)=e^{-i\Delta(H_D-\lambda C_n)},
\qquad
c_\lambda=\lambda(2-\lambda),
```

define

```math
\mathcal K_{a,\lambda}[N]
:=
a\sum_nN(an)A_{n,\lambda}^{(1)},
\qquad
\mathfrak C_{a,\lambda}[N,M]
:=
[\mathcal K_{a,\lambda}[N],\mathcal K_{a,\lambda}[M]].
```

Then

```math
c_\lambda^{-2}\mathfrak C_{a,\lambda}[N,M]\iota_a\phi
=
\iota_a K_\parallel[\beta]\phi+O(a).
```

This is the first rigorous stochastic-curvature statement in the V2 sense: the continuum tangential hypersurface-deformation operator is recovered from the exact finite ISP exchange-coefficient algebra.

## 2. Exact Inputs

The proof uses only the following exact free-core facts.

### Input A: leading collar-excision coefficient

For singleton support `R_n={n}`, the collar-excision comparison map has

```math
J_n(\Delta)=I+\Delta^2A_n^{(1)}+O(\Delta^4),
```

where `A_n^{(1)}` is supported on `N_1({n})`, has column sums zero, and has the explicit deleted-bond entries

```math
[A_n^{(1)}]_{(n,s),(n,s)}=\frac{1}{2a^2},
```

```math
[A_n^{(1)}]_{(n\pm1,\bar s),(n,s)}
=
[A_n^{(1)}]_{(n,s),(n\pm1,\bar s)}
=
-\frac{1}{4a^2},
```

```math
[A_n^{(1)}]_{(n\pm1,s),(n\pm1,s)}=\frac{1}{4a^2},
```

with all other entries zero.

### Input B: leading exchange coefficient

For singleton supports whose first visible exchange order is generated by leading coefficients,

```math
E_{n,m}(\Delta)=I+\Delta^4[A_n^{(1)},A_m^{(1)}]+O(\Delta^6).
```

In particular, the coefficient-level object behind the first visible exchange defect is the commutator of the leading comparison coefficients.

### Input C: bond-centered singleton thin-slab identity

The exact singleton commutator algebra closes on the bond-flip and two-step strip channels. After bond-centered spatial renormalization, its action on smooth sampled spinors is

```math
[\mathcal K_a[N],\mathcal K_a[M]]\iota_a\phi
=
\iota_a\left(\left(\beta\partial_x+\frac12\partial_x\beta\right)(I-\alpha)\phi\right)
+O(a).
```

This is the exact bond-centered collar-excision thin-slab theorem from the free-model sequel. The present document repackages it as the coefficient-level stochastic-curvature theorem and adds the normalized `lambda`-family consequence.

### Input D: exact `lambda` scaling

For every singleton support and every `0<lambda<=1`,

```math
A_{n,\lambda}^{(1)}=c_\lambda A_n^{(1)},
\qquad
c_\lambda=\lambda(2-\lambda).
```

The same identity also holds for finite interval supports at leading order, but the theorem below only needs the singleton case.

## 3. Why This Is The Right Coefficient Object

The finite exchange defect is a group commutator,

```math
E_{R,S}(\Delta)=J_RJ_SJ_R^{-1}J_S^{-1}.
```

At the first visible singleton order,

```math
J_n=I+\Delta^2A_n^{(1)}+O(\Delta^4),
```

so the group commutator gives

```math
E_{n,m}(\Delta)
=
I+\Delta^4[A_n^{(1)},A_m^{(1)}]+O(\Delta^6).
```

Thus the infinitesimal exchange curvature is not `A_n^{(1)}` itself. It is the antisymmetric bilinear commutator of two localized leading deformation coefficients.

For smooth lapse profiles, the natural coefficient-level smeared deformation is therefore

```math
\mathcal K_a[N]=a\sum_nN_nA_n^{(1)},
\qquad
N_n=N(an),
```

and the coefficient-level curvature is

```math
\mathfrak C_a[N,M]
=
[\mathcal K_a[N],\mathcal K_a[M]]
=
a^2\sum_{n,m}N_nM_m[A_n^{(1)},A_m^{(1)}].
```

This is the coefficient that a smooth-lapse finite-slab theorem must recover before remainders are considered.

## 4. Proof Of The Collar-Excision Theorem

We prove

```math
\mathfrak C_a[N,M]\iota_a\phi
=
\iota_a K_\parallel[\beta]\phi+O(a).
```

### Step 1: localize the commutator sum

Because `A_n^{(1)}` is supported on `N_1({n})`, the commutator `[A_n^{(1)},A_m^{(1)}]` vanishes unless the one-step neighborhoods overlap. Hence only finitely many relative displacements `r=m-n` contribute:

```math
\mathfrak C_a[N,M]
=
a^2\sum_n\sum_{r\in R_*}N_nM_{n+r}[A_n^{(1)},A_{n+r}^{(1)}],
```

where `R_*` is the finite active displacement set. In the singleton leading collar-excision case, this is the adjacent and next-nearest exchange sector, with the known Dirac parity split: odd displacement gives spin-flip strips and even displacement gives same-spin strips.

### Step 2: antisymmetrize the lapse factor

The commutator is antisymmetric under exchanging the two localized deformations. Therefore the sum can be written in terms of

```math
N_nM_{n+r}-M_nN_{n+r}.
```

For smooth `N,M`, Taylor expansion at the bond-centered point gives, uniformly on the finite active displacement set,

```math
N_nM_{n+r}-M_nN_{n+r}
=
ar\,\beta(x_{n+r/2})+O(a^2),
```

with

```math
\beta=N\partial_xM-M\partial_xN.
```

This is the first place the hypersurface-deformation structure appears: the antisymmetry of exchanging normal deformations produces the tangential lapse `beta`.

### Step 3: use the exact strip closure

The explicit singleton calculations show that the active commutators close on finitely many bond-centered strip channels. Schematically,

```math
[A_n^{(1)},A_{n+r}^{(1)}]
=
\sum_\chi c_\chi(r)X_{n,r,\chi},
```

where the `X_{n,r,chi}` are the exact antisymmetric bond-flip and two-step strip operators and the `c_chi(r)` are finitely supported exact coefficients. The exact computations give finite first moments

```math
C_\chi=\sum_r r\,c_\chi(r).
```

Substituting the Taylor expansion from Step 2 turns the commutator sum into a finite linear combination of bond-centered strip densities:

```math
\mathfrak C_a[N,M]\iota_a\phi
=
\sum_\chi C_\chi\,\mathcal T_{a,\chi}[\beta]\iota_a\phi+O(a).
```

Here each `T_{a,chi}[beta]` is an exact bond-centered difference operator obtained from the transition-law coefficient algebra. No entrywise matrix limit is being taken.

### Step 4: take the bond-centered profile limit

The bond-centered thin-slab computation evaluates the finite strip moments and the spin/parity action of the active channels. Its exact conclusion is

```math
\sum_\chi C_\chi\,\mathcal T_{a,\chi}[\beta]\iota_a\phi
=
\iota_a\left(\left(\beta\partial_x+\frac12\partial_x\beta\right)(I-\alpha)\phi\right)
+O(a).
```

Combining Steps 1 through 4 gives

```math
\mathfrak C_a[N,M]\iota_a\phi
=
\iota_a K_\parallel[\beta]\phi+O(a),
```

which is the claimed coefficient-level stochastic-curvature theorem. Taking `a->0` gives the strong limit on sampled smooth compactly supported profiles.

This proves the collar-excision case.

## 5. Proof Of The `lambda`-Family Theorem

For the `lambda` family, the exact leading coefficient identity is

```math
A_{n,\lambda}^{(1)}=c_\lambda A_n^{(1)},
\qquad
c_\lambda=\lambda(2-\lambda).
```

Therefore

```math
\mathcal K_{a,\lambda}[N]
=
a\sum_nN_nA_{n,\lambda}^{(1)}
=
c_\lambda a\sum_nN_nA_n^{(1)}
=
c_\lambda\mathcal K_a[N].
```

Taking commutators gives the exact finite-lattice identity

```math
\mathfrak C_{a,\lambda}[N,M]
=
[\mathcal K_{a,\lambda}[N],\mathcal K_{a,\lambda}[M]]
=
c_\lambda^2[\mathcal K_a[N],\mathcal K_a[M]]
=
c_\lambda^2\mathfrak C_a[N,M].
```

Hence

```math
c_\lambda^{-2}\mathfrak C_{a,\lambda}[N,M]
=
\mathfrak C_a[N,M].
```

Applying the collar-excision theorem just proved yields

```math
c_\lambda^{-2}\mathfrak C_{a,\lambda}[N,M]\iota_a\phi
=
\iota_a K_\parallel[\beta]\phi+O(a).
```

This proves normalized regulator independence for the exact `lambda` family at the leading coefficient level.

## 6. Relation To The Finite Exchange Defect

The theorem above is coefficient-level. It identifies the continuum limit of the first visible exchange coefficient.

For a finite-slab theorem one would need to prove, for a smooth-lapse finite deformation rule,

```math
Z_{a,\Delta}\bigl(E_{N,M}(a,\Delta)-I\bigr)\iota_a\phi
\to
\iota_a K_\parallel[\beta]\phi.
```

The coefficient theorem proves the target of the leading term. It does not by itself control the remainder

```math
O(\Delta^6)
```

after lattice renormalization as `a->0`. Because the free Dirac Hamiltonian scales like `a^{-1}`, such a finite-slab theorem needs a coupled small-slab condition on `Delta(a)` and uniform bounds on higher coefficients.

Thus the result proved here is exactly the first theorem needed before the finite-slab lift.

## 7. What Is Now Proved

The following statements are now theorem-level at the stated scope.

1. The exact leading ISP exchange coefficient has a bond-centered continuum action.
2. That action is the tangential operator

```math
K_\parallel[N\partial_xM-M\partial_xN]
=
\left((N\partial_xM-M\partial_xN)\partial_x
+\frac12\partial_x(N\partial_xM-M\partial_xN)\right)(I-\alpha).
```

3. The result is not an entrywise matrix limit; it is a strong profile-action limit on smooth sampled spinors.
4. The exact `lambda` family shares the same limit after division by `c_lambda^2`.
5. No finite-slab remainder theorem, all-support theorem, all-order regulator theorem, Fock-space theorem, gauge theorem, or gravity theorem is implied.

## 8. Immediate Next Theorem

The immediate log-smeared finite-slab lift is now proved in `finite-slab-free-stochastic-curvature-theorem.md`. The next broader theorem should replace log-smearing by a primitive weighted-Hamiltonian smooth-lapse kernel, or weaken the very-thin-slab scaling if sharper remainder estimates permit it.

The target finite-slab reduction has the form:


```math
\frac{1}{c_\lambda^2\Delta^4}
\bigl(E_{N,M}^{(\lambda)}(a,\Delta)-I\bigr)\iota_a\phi
=
\mathfrak C_a[N,M]\iota_a\phi
+\text{controlled remainder},
```

with an explicit condition on `Delta(a)` that makes the remainder vanish after the spatial normalization. The coefficient theorem above supplies the limiting operator; the finite-slab theorem must supply the missing uniform estimate.
