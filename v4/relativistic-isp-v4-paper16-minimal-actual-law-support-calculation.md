## Abstract

Paper 15 ended with the minimal rectangle-fiber mass vector

$$
\boxed{
\left(m^H_{\ell,u_0},m^H_{\ell,u_1}\right),
\qquad
m^H_{\ell,u}:=
\mathbb P_a^{act}(U=u\mid H=\ell),
}
$$

or equivalently with the unnormalized masses

$$
\boxed{
\widehat m^H_{\ell,u}:=
\mathbb P_a^{act}(H=\ell,U=u).
}
$$

The current corpus did not prove either positivity

$$
\widehat m^H_{\ell,u}>0
$$

or exclusion

$$
\widehat m^H_{\ell,u}=0
$$

for the two live row values \(u_0,u_1\).

Paper 16 asks whether the known operational write-register theorem can fill
this table.  It cannot, except under an explicit base-support hypothesis.  It
proves the exact finite calculation:

$$
\boxed{
\mathbb P^{wr}(H=\ell,U=u,O=o)
=
D(o\mid u)\,\mathbb P^{act}(H=\ell,U=u).
}
$$

Thus a detector with \(D(o\mid u)>0\) preserves positivity equivalently:

$$
\boxed{
\mathbb P^{wr}(H=\ell,U=u,O=o)>0
\quad\Longleftrightarrow\quad
\mathbb P^{act}(H=\ell,U=u)>0.
}
$$

The instrument is real, finite, positive, and Barandes-aligned as an
operational record.  But it is not an actual-law support source unless the
pre-instrument row mass is already sourced.

Fix a finite total-record space \(\Omega_a\) with actual law

\boxed{

\qquad

$$

Assume the formal admissible row fiber has exactly two values:

$$
\boxed{
{\mathcal A}^{NN\mid H}_{a,ij}(\ell)=\{u_0,u_1\}.

$$

The actual row mass table is

$$
\boxed{
\begin{array}{c|cc}

\widehat m_0&

with

$$
\boxed{
\widehat m_b
:=
\mathbb P_a^{act}(H=\ell,U=u_b),

$$

The normalized conditional masses are

$$
\boxed{
m_b
:=
\mathbb P_a^{act}(U=u_b\mid H=\ell)
=
{\widehat m_b\over \widehat m_0+\widehat m_1}

$$

if no other actual row values occur in the rectangle fiber.  In the general
case one divides by \(\mathbb P_a^{act}(H=\ell)\), but the two-row live table
is the first nontrivial obstruction.

### Definition 1.1: Support Outcomes

The table has:

1. **Feynman support** if

$$
\boxed{
\widehat m_0>0,\qquad \widehat m_1>0.
}
$$

2. **Einstein support selection** if exactly one of the two masses is

$$
\boxed{
\widehat m_b>0,\qquad \widehat m_{1-b}=0,
\qquad
\hbox{with a row-sensitive support-kernel explanation.}
}
$$

3. **Unsourced support** if the current data specify the formal row set but
do not specify which masses are positive.

The problem is not to name one of these outcomes.  The problem is to compute
or source it from the actual law.

## 2. Feynman Calculation: The Write-Register Instrument

Feynman's move is to attach a detector to the smallest live table and ask
what mass comes out.

Let \(O=\{o_0,o_1\}\) be a detector register.  Let the detector read the row
through a stochastic response matrix

$$
\boxed{
D(o_c\mid u_b)\ge0,
\qquad
\sum_{c=0}^1D(o_c\mid u_b)=1.
}
$$

Define the nondisturbing row detector by

$$
\boxed{
M_{o_c}(y\mid x)
=
D(o_c\mid U(x))\,\delta_{y,x}.
}
$$

This is a V2 write-register detector: it writes a record while leaving the
system coordinate unchanged.

### Theorem 2.1: Exact Row-Record Mass Formula

For the nondisturbing row detector,

$$
\boxed{
\mathbb P^{wr}(H=\ell,U=u_b,O=o_c)
=
D(o_c\mid u_b)\,\widehat m_b.
}
$$

Consequently, if \(D(o_c\mid u_b)>0\), then

$$
\boxed{
\mathbb P^{wr}(H=\ell,U=u_b,O=o_c)>0
\quad\Longleftrightarrow\quad
\widehat m_b>0.
}
$$

Proof.  The nondisturbing detector multiplies the actual mass at \(x\) by
\(D(o_c\mid U(x))\) and does not change \(H(x)\) or \(U(x)\).  Summing over
all \(x\in\Omega_a\) with \(H(x)=\ell\) and \(U(x)=u_b\) gives

$$
\begin{aligned}
\mathbb P^{wr}(H=\ell,U=u_b,O=o_c)
&=
\sum_{\substack{x:H(x)=\ell\\ U(x)=u_b}}
D(o_c\mid U(x))\,\mathbb P_a^{act}(x)
\\
&=
D(o_c\mid u_b)
\sum_{\substack{x:H(x)=\ell\\ U(x)=u_b}}
\mathbb P_a^{act}(x)
\\
&=
D(o_c\mid u_b)\,\widehat m_b.
\end{aligned}
$$

The positivity equivalence follows because \(D(o_c\mid u_b)>0\).  `square`

### Corollary 2.2: Instruments Reveal Support But Do Not Create It

A positive row detector can reveal that a row has positive actual mass, but
it cannot turn a zero row into a positive row:

$$
\boxed{
\widehat m_b=0
\quad\Longrightarrow\quad
\mathbb P^{wr}(H=\ell,U=u_b,O=o_c)=0
\quad\hbox{for all }o_c.
}
$$

Thus the V2 write-register theorem is not by itself a proof of
rectangle-row positivity.  It is a support-preserving readout theorem.

### Corollary 2.3: Perfect Readout Does Not Solve Base Support

For the perfect detector

$$
\boxed{
D(o_c\mid u_b)=\delta_{bc},
}
$$

one has

$$
\boxed{
\mathbb P^{wr}(H=\ell,O=o_b)=\widehat m_b.
}
$$

Therefore a perfect row detector gives the cleanest possible experiment:
measure the row and read off the mass.  But the mass is still the
pre-instrument actual mass.  The detector computes it; it does not source it.

## 3. Einstein Calculation: Constraint And Ward Exclusion

Einstein's move is different.  If a row has zero mass, the zero should be
explained by an invariant or constraint, not by arbitrary absence.

Let

$$
\boxed{
C:\Omega_a\to{\mathcal C}_a
}
$$

be a finite actual constraint record.  Suppose actual support obeys

$$
\boxed{
\mathbb P_a^{act}(C=0)=1.
}
$$

### Definition 3.1: Row-Sensitive Constraint

The constraint \(C\) is row-sensitive on the rectangle fiber \(\ell\) if
there is a function

$$
\boxed{
c_\ell:\{u_0,u_1\}\to{\mathcal C}_a
}
$$

such that on \(H=\ell\),

$$
\boxed{
C=c_\ell(U)
}
$$

on actual support or on the formal row body being tested.

It excludes \(u_b\) if

$$
\boxed{
c_\ell(u_b)\ne0.
}
$$

### Theorem 3.2: Constraint Exclusion Criterion

If \(C=0\) almost surely and \(C=c_\ell(U)\) on the row body over \(H=\ell\),
then

$$
\boxed{
c_\ell(u_b)\ne0
\quad\Longrightarrow\quad
\widehat m_b=0.
}
$$

If \(c_\ell(u_0)=c_\ell(u_1)=0\), the constraint excludes neither row.

Proof.  If \(H=\ell\) and \(U=u_b\) occur with positive mass, then on that
event \(C=c_\ell(u_b)\).  Since \(C=0\) almost surely, a nonzero
\(c_\ell(u_b)\) makes the event impossible.  If both row values map to zero,
the constraint is silent on the two-row support question.  `square`

### Corollary 3.3: Hyp-Only Ward Identities Are Row-Blind

If \(C\) factors through the rectangle or hyp record alone,

$$
\boxed{
C=\Psi(H),
}
$$

then on a fixed fiber \(H=\ell\),

$$
\boxed{
c_\ell(u_0)=c_\ell(u_1)=\Psi(\ell).
}
$$

Thus \(C\) cannot select exactly one of \(u_0,u_1\).  It either kills the
whole rectangle fiber or kills neither row.

This is the exact version of the Paper-15 warning: hyp-only Ward defects
cannot produce singleton NN row support.

## 5. Why This Is Not Another Circle

Paper 16 removes two tempting but false exits.

### False Exit 5.1: "Instrument Support Solves It"

No.  The instrument solves the **readout** problem:

$$
\boxed{
\hbox{if a row has mass, a licensed detector can record it.}
}
$$

It does not solve the **base support** problem:

$$
\boxed{
\hbox{which rows have mass before the detector is attached?}
}
$$

This is exactly aligned with the V2 operational no-go: one primitive endpoint
kernel does not determine the operational family, and one detector family
does not determine support absent the base law.

### False Exit 5.2: "Ward Support Solves It"

No.  A Ward identity solves the row problem only if it is row-sensitive.
Gauge/Haar, hyp-only, rectangle-only, or label-only identities are row-blind
inside the fixed fiber.

The needed object is not the word "Ward."  It is the finite table

$$
\boxed{
c_\ell(u_0),\quad c_\ell(u_1).
}
$$

with exactly one nonzero entry, or with both zero plus a separate positivity
theorem.

## 7. Einstein Route: Licensed Support-Kernel Principle

The Einstein move is not to add another detector.  It is to demand a
principle of sufficient reason for zero rows.

On the two-row rectangle fiber, a row should have zero actual mass only
because some licensed finite record or finite constraint forbids it.  The
support should not contain unexplained absences.

### Definition 7.1: Licensed Support-Kernel Principle

Fix a positive-mass rectangle fiber \(\ell\) with formal row body

$$
\boxed{
{\mathcal A}^{NN\mid H}_{a,ij}(\ell)=\{u_0,u_1\}.
}
$$

A licensed support-kernel explanation consists of a finite actual record or
constraint

$$
\boxed{
C^{supp}_{a,ij}:\Omega_a\to{\mathcal C}_a
}
$$

and a row table

$$
\boxed{
c_\ell:\{u_0,u_1\}\to{\mathcal C}_a
}
$$

such that:

1. **actual constraint**

$$
\boxed{
\mathbb P_a^{act}(C^{supp}_{a,ij}=0)=1;
}
$$

2. **row factorization on the tested body**

$$
\boxed{
C^{supp}_{a,ij}=c_\ell(U)
\quad\hbox{on }H=\ell\hbox{ and the tested row body};
}
$$

3. **noncircular licensing**: \(C^{supp}_{a,ij}\) is built from declared
finite records, Ward identities, conservation constraints, residual records,
or operationally declared variables before the support question is asked; it
is not defined as \(1_{\{\widehat m_b=0\}}\);

4. **no accidental zeros**

$$
\boxed{
c_\ell(u_b)=0
\quad\Longrightarrow\quad
\widehat m_b>0.
}
$$

The zero set

$$
\boxed{
Z_\ell(C):=\{u_b:c_\ell(u_b)=0\}
}
$$

is then the actual support inside the formal row body:

$$
\boxed{
{\mathcal S}^{NN\mid H}_{a,ij}(\ell)
=
{\mathcal A}^{NN\mid H}_{a,ij}(\ell)\cap Z_\ell(C).
}
$$

This is the Einstein principle in finite form:

$$
\boxed{
\hbox{actual zeros require licensed finite reasons, and licensed allowed
rows are not accidentally absent.}
}
$$

### Theorem 7.2: Support-Kernel Trichotomy

Assume the licensed support-kernel principle on the two-row fiber.  Then
exactly one of the following holds:

1. \(Z_\ell(C)=\{u_0,u_1\}\).  Then

$$
\boxed{
\widehat m_0>0,\qquad \widehat m_1>0,
}
$$

so the Feynman two-row support route holds.

2. \(Z_\ell(C)=\{u_b\}\) for one \(b\in\{0,1\}\).  Then

$$
\boxed{
\widehat m_b>0,\qquad \widehat m_{1-b}=0,
}
$$

so the Einstein singleton-selection route holds.

3. \(Z_\ell(C)=\varnothing\).  Then

$$
\boxed{
\mathbb P_a^{act}(H=\ell)=0,
}
$$

contradicting the live positive-mass rectangle fiber.

Proof.  If \(c_\ell(u_b)\ne0\), Theorem 3.2 gives
\(\widehat m_b=0\).  If \(c_\ell(u_b)=0\), the no-accidental-zeros clause
gives \(\widehat m_b>0\).  The three cases are the possible zero sets inside
\(\{u_0,u_1\}\).  If the zero set is empty, both row masses vanish, so the
rectangle fiber has no actual mass, contrary to the live-fiber hypothesis.
`square`

### Corollary 7.3: What Einstein Needs

The Einstein route is now a single finite table:

$$
\boxed{
c_\ell(u_0),\qquad c_\ell(u_1).
}
$$

To get singleton row selection, the table must have exactly one zero:

$$
\boxed{
c_\ell(u_b)=0,\qquad c_\ell(u_{1-b})\ne0.
}
$$

To get the Feynman floor instead, the same principle allows both rows:

$$
\boxed{
c_\ell(u_0)=0,\qquad c_\ell(u_1)=0.
}
$$

Thus the Einstein route does not compete with Feynman's calculation.  It
explains which rows Feynman's detector should see.
