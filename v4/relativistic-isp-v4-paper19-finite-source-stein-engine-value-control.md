### Definition 2.2: Licensed Finite Stein Generator

\boxed{

\boxed{

The generator is \(\pi_a\)-reversible when

$$
\boxed{

For a function class \({\mathcal F}_a\), the actual Stein defect is

\boxed{

## 3. The Exact Finite Stein Identity

### Theorem 3.1: Reversible Stein Identity

If \(K_a\) is \(\pi_a\)-reversible, then

$$
\boxed{
\mathbb E_{\pi_a}[K_af]=0
\qquad\hbox{for every }f:L_a\to\mathbb R.
}
$$

Proof.  By reversibility,

$$
\sum_q\pi_a(q)K_af(q)
=
\sum_{q,r}\pi_a(q)k_a(q,r)(f(r)-f(q)).
$$

Swap \(q,r\) in the first half and use
\(\pi_a(q)k_a(q,r)=\pi_a(r)k_a(r,q)\).  The sum cancels pairwise.
\(\square\)

### Theorem 3.2: Source-Stein Value Control

Let \(V_a:L_a\to\mathbb R\) be a centered target relative to \(\pi_a\):

$$
\boxed{
\mathbb E_{\pi_a}V_a=0.
}
$$

Assume the finite Poisson equation

$$
\boxed{
-K_af_a=V_a
}
$$

has a solution \(f_a\in{\mathcal F}_a\).  Then

$$
\boxed{
\left|\mathbb E_{P_a}V_a\right|
\le
\|f_a\|_{{\mathcal F}_a}\Delta_a({\mathcal F}_a).
}
$$

Proof.  Since \(V_a=-K_af_a\),

$$
\mathbb E_{P_a}V_a=-\mathbb E_{P_a}K_af_a.
$$

The definition of \(\Delta_a\) gives the bound. \(\square\)

### Corollary 3.3: Calibrated Value Transfer

For general \(V_a\),

$$
\boxed{
\left|
\mathbb E_{P_a}V_a-\mathbb E_{\pi_a}V_a
\right|
\le
\|f_a\|_{{\mathcal F}_a}\Delta_a({\mathcal F}_a)
}
$$

whenever

$$
\boxed{
-K_af_a
=
V_a-\mathbb E_{\pi_a}V_a.
}
$$

Thus same-law value control reduces to:

1. source visibility of \(V_a\);
2. a licensed reversible generator \(K_a\);
3. solvability and norm control for the Poisson equation;
4. an actual Stein defect bound.

This is the E2+E3 engine.

## 4. Coercivity And Spectral Gap

The Poisson solution is useful only if its norm is controlled.

### Definition 4.1: Finite Dirichlet Form

For reversible \(K_a\), define

$$
\boxed{
{\mathcal E}_a(f,f)
=
\frac12
\sum_{q,r}
\pi_a(q)k_a(q,r)(f(r)-f(q))^2.
}
$$

### Definition 4.2: Finite Gap

The finite gap is the largest \(\lambda_a>0\) such that

$$
\boxed{
{\mathcal E}_a(f,f)
\ge
\lambda_a\,\operatorname{Var}_{\pi_a}(f)
}
$$

for all \(f\).

### Theorem 4.3: Gap Gives Poisson Control

Assume \(K_a\) is irreducible on \(L_a\) and has gap \(\lambda_a>0\).  If
\(\mathbb E_{\pi_a}V_a=0\), then the Poisson equation

$$
\boxed{
-K_af_a=V_a
}
$$

has a centered solution, and

$$
\boxed{
\|f_a\|_{L^2(\pi_a)}
\le
\lambda_a^{-1}\|V_a\|_{L^2(\pi_a)}.
}
$$

Proof.  On the centered subspace, \(-K_a\) is positive self-adjoint with
spectrum bounded below by \(\lambda_a\).  Invert it there. \(\square\)

### Corollary 4.4: Quantitative E2+E3 Bound

If the Stein defect is measured in \(L^2(\pi_a)\)-dual norm, then

$$
\boxed{
\left|
\mathbb E_{P_a}V_a-\mathbb E_{\pi_a}V_a
\right|
\le
\lambda_a^{-1}\|V_a-\mathbb E_{\pi_a}V_a\|_{L^2(\pi_a)}
\Delta_a.
}
$$

More carefully, the last factor is the defect applied to the normalized
Poisson solution.  The important lesson is:

$$
\boxed{
\hbox{value control requires a gap or an equivalent coercive estimate.}

## 7. Ward/Stein Coercivity

\boxed{

## 60. Minimal Non-Tautological Value Extension

We now attack the law-selection theorem:

$$
\boxed{
\mathrm{V4P19\text{-}MINIMAL\text{-}NONTAUTOLOGICAL\text{-}VALUE\text{-}EXTENSION}.
}
$$

The question is not whether least bias is a useful choice.  The question is
whether the listed consistency requirements force it.

### Definition 60.1: Finite Admissible Update Rule

Let \(Q\) be a finite hard-live configuration set and let the positive
reference law satisfy

$$
\boxed{
\mu^0(q)>0\qquad(q\in Q).
}
$$

A source update rule \({\mathcal U}\) assigns to every licensed finite
source constraint family

$$
\boxed{
{\mathcal C}
=
\left\{
P\in{\mathcal P}(Q):
\mathbb E_P O_k=m_k,\quad k=1,\ldots,r
\right\}
}
$$

a unique posterior

$$
\boxed{
P^*={\mathcal U}(Q,\mu^0,{\mathcal C})\in{\mathcal C}.
}
$$

The rule is admissible if it satisfies:

$$
\boxed{
\begin{array}{ll}
\mathrm{A1}:&\hbox{hard support: }P^*(q)=0\hbox{ outside }H,\\
\mathrm{A2}:&\hbox{positive reference: }\mu^0(q)>0\hbox{ on }H,\\
\mathrm{A3}:&\hbox{licensed source use: only licensed }O_k,m_k\hbox{ enter},\\
\mathrm{A4}:&\hbox{source-local coarse-graining consistency},\\
\mathrm{A5}:&\hbox{independence/product composition},\\
\mathrm{A6}:&\hbox{no posterior row-selector fitting}.
\end{array}
}
$$

The key interpretive issue is A4.  Weak coarse-graining is not enough.  The
needed finite version is source-locality:

$$
\boxed{
\hbox{if no licensed source distinguishes two states inside a block,
the update preserves their prior conditional odds.}
}
$$

Thus, for a source value \(s(q)\), the posterior-to-prior ratio may depend on
\(q\) only through licensed source values:

$$
\boxed{
\frac{P^*(q)}{\mu^0(q)}=
G_\lambda(s(q)).
}
$$

This is the precise non-tautological content of "ignore distinctions not
licensed by the source."

### Lemma 60.2: Source Locality Gives A Local Multiplier Form

Assume A1--A4 and finite regularity of the update as the target varies.
For a scalar licensed source

$$
\boxed{
O(q)=x_q,
}
$$

there is a positive multiplier

$$
\boxed{
g_\lambda
}
$$

such that

$$
\boxed{
P^*(q)=\frac{\mu^0(q)g_\lambda(x_q)}
{\sum_{q'}\mu^0(q')g_\lambda(x_{q'})}.
}
$$

For several licensed sources,

$$
\boxed{
O(q)=\bigl(O_1(q),\ldots,O_r(q)\bigr),
}
$$

the same statement holds with a positive multiplier on the licensed source
vector:

$$
\boxed{
P^*(q)=\frac{\mu^0(q)g_\lambda(O(q))}
{\sum_{q'}\mu^0(q')g_\lambda(O(q'))}.
}
$$

Only after product composition is imposed in Lemma 60.3 does this multiplier
become exponential:

$$
\boxed{
g_\lambda(O(q))=\exp\{\lambda\cdot O(q)\}.
}
$$

Proof.  If two states have the same licensed source value, A4 says the
update cannot change their conditional prior odds:

$$
\frac{P^*(q)}{P^*(q')}
=
\frac{\mu^0(q)}{\mu^0(q')}
\qquad
\hbox{whenever }O(q)=O(q').
$$

Therefore

$$
\boxed{
\frac{P^*(q)}{\mu^0(q)}
}
$$

is constant on source level sets.  Call that constant

$$
\boxed{
g_\lambda(O(q)).
}
$$

Normalization gives the displayed form.  The parameter

$$
\boxed{
\lambda
}
$$

labels the licensed target

$$
\boxed{
m.
}
$$

Finite regularity rules out discontinuous target-dependent relabelings of
the level sets.  This completes the proof.

### Lemma 60.3: Product Composition Forces Exponential Multipliers

Assume A5.  Let two independent systems have references

$$
\boxed{
\mu_A,\mu_B,
}
$$

source values

$$
\boxed{
x_i,\ y_j,
}
$$

and separated source constraints.  The joint reference is

$$
\boxed{
\mu_{AB}(i,j)=\mu_A(i)\mu_B(j),
}
$$

and the joint source is additive:

$$
\boxed{
O_{AB}(i,j)=O_A(i)+O_B(j).
}
$$

Then the multiplier in Lemma 60.2 must be exponential:

$$
\boxed{
g_\lambda(t)=e^{c\lambda t},
}
$$

for a fixed nonzero scale \(c\), absorbable into \(\lambda\).

Proof.  By Lemma 60.2, the joint update has the form

$$
P^*_{AB}(i,j)
\propto
\mu_A(i)\mu_B(j)g_{\lambda_A,\lambda_B}(x_i+y_j).
$$

A5 requires the same update to equal the product of the separate updates:

$$
P^*_{AB}(i,j)
=
P^*_A(i)P^*_B(j)
\propto
\mu_A(i)\mu_B(j)g_{\lambda_A}(x_i)g_{\lambda_B}(y_j).
$$

Therefore, up to a normalization constant independent of \(i,j\), we have

$$
\boxed{
g(x+y)=C\,g(x)g(y).
}
$$

Normalize \(g(0)=1\).  Then \(C=1\).  Finite regularity gives the regular
Cauchy solution:

$$
\boxed{
g(t)=e^{ct}.
}
$$

Changing the scale of the multiplier parameter absorbs \(c\). \(\square\)

### Theorem 60.4: Minimal Non-Tautological Extension Is Least Bias

Under A1--A6, with A4 read as source-locality and with finite regularity,
the only admissible finite update is:

$$
\boxed{
P^*
=
\arg\min_{P\in{\mathcal C}}D(P\|\mu^0).
}
$$

Equivalently, for licensed linear source constraints,

$$
\boxed{
P^*(q)
=
\frac{\mu^0(q)\exp\{\lambda\cdot O(q)\}}
{\sum_{q'}\mu^0(q')\exp\{\lambda\cdot O(q')\}},
}
$$

where \(\lambda\) is chosen so that

$$
\boxed{
\mathbb E_{P^*}O_k=m_k.
}
$$

Proof.  Lemma 60.2 reduces every admissible update to a local
posterior-to-prior multiplier depending only on licensed source values.
Lemma 60.3 uses product composition to force that multiplier to be
exponential.  The displayed exponential family is exactly the Lagrange
multiplier solution of

$$
\boxed{
\min_{P\in{\mathcal C}}\sum_q P(q)\log\frac{P(q)}{\mu^0(q)}.
}
$$

Indeed, differentiating

$$
\sum_qP(q)\log\frac{P(q)}{\mu^0(q)}
-\alpha\left(\sum_qP(q)-1\right)
-\sum_k\lambda_k\left(\sum_qP(q)O_k(q)-m_k\right)
$$

gives

$$
\boxed{
\log\frac{P(q)}{\mu^0(q)}+1-\alpha-\lambda\cdot O(q)=0,
}
$$

hence

$$
\boxed{
P(q)\propto\mu^0(q)e^{\lambda\cdot O(q)}.
}
$$

Strict convexity of

$$
\boxed{
D(P\|\mu^0)
}
$$

on the positive simplex gives uniqueness whenever the constraint set is
feasible.  A6 forbids choosing \(O\) or \(m\) after seeing the desired row
ratio, so the result is non-tautological. \(\square\)

### Feynman Counterexample 60.5: Why A4/A5 Cannot Be Weakened

If A4 is weakened to ordinary support-respecting coarse labels, the theorem
fails.  For example, consider the alternative update:

$$

=
\arg\min_{P\in{\mathcal C}}
\left[

+\epsilon\sum_q\mu^0(q)
\left(\frac{P(q)}{\mu^0(q)}-1\right)^4
\right].
}
$$

For

$$
\boxed{

}
$$

this rule:

$$
\boxed{
\begin{array}{ll}
\hbox{respects hard support},\\
\hbox{uses the positive reference law},\\
\hbox{uses the licensed constraints},\\
\hbox{does not explicitly choose a row selector}.
\end{array}
}
$$

But it is not the KL update.  It fails the strong form of source-locality and
product composition: for independent systems, the quartic correction produces
cross terms that do not factor into separate updates.

Thus the Feynman test identifies the missing axiom:

$$
\boxed{
\hbox{without strong source-local coarse-graining and product composition,
least bias is not forced.}
}
$$
