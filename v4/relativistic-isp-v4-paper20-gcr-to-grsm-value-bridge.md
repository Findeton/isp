## 3. The Bridge Graph

Let \({\mathcal E}\) be a finite list of live two-row fibers.  Each edge

$$
\boxed{

}
$$

has two rows:

$$
\boxed{

Route A supplies an edge log-ratio:

$$
\boxed{

Route B supplies a readout:

$$
\boxed{

\qquad

}
$$

The bridge asks whether the GR/SM log-ratio is generated by an operational
potential on readout states.

### Definition 3.1: Potential Bridge

A potential bridge is a prelicensed finite function:

$$
\boxed{
\Phi:{\mathcal Y}\to{\mathbb R}
}
$$

such that every edge satisfies:

$$
\boxed{
L_A(e)
=
\Phi(y_e^+)-\Phi(y_e^-).
}
$$

If a reference density correction is needed, replace \(L_A(e)\) by:

$$
\boxed{
\widetilde L_A(e)
:=
L_A(e)
-
\log
\frac{\rho_{e,+}^{op}}{\rho_{e,-}^{op}}.
}
$$

Then the bridge condition is:

$$
\boxed{
\widetilde L_A(e)
=
\Phi(y_e^+)-\Phi(y_e^-).
}
$$

This correction lets the operational reference measure carry part of the
weight, while the potential carries the source-response part.

## 4. Exactness Tests

The potential bridge is a finite exact one-form problem.  This gives hard
Feynman tests.

### Theorem 4.1: Bridge Potential Exists If And Only If Cycle Sums Vanish

Let \(G_Y\) be the directed graph whose vertices are readout states and whose
edges are:

$$
\boxed{
y_e^-\to y_e^+.
}
$$

Assign each edge the corrected GR/SM log-ratio:

$$
\boxed{
\widetilde L_A(e).
}
$$

There exists a potential bridge:

$$
\boxed{
\widetilde L_A(e)=\Phi(y_e^+)-\Phi(y_e^-)
}
$$

if and only if every directed cycle \(C\) satisfies:

$$
\boxed{
\sum_{e\in C}\sigma_C(e)\widetilde L_A(e)=0,
}
$$

where \(\sigma_C(e)=+1\) when the cycle follows the edge orientation and
\(\sigma_C(e)=-1\) when it traverses the edge backward.

Proof.  This is the finite exactness criterion for a one-cochain on a graph.
If \(\widetilde L_A=d\Phi\), every cycle sum telescopes to zero.  Conversely,
if all cycle sums vanish, define \(\Phi\) at one vertex in each connected
component and extend it along paths.  Vanishing cycle sums make the value
path-independent. \(\square\)

### Corollary 4.2: Repeated-Edge Falsifier

If two edges have the same ordered readout pair:

$$
\boxed{
(y_e^-,y_e^+)=(y_f^-,y_f^+),
}
$$

but different corrected log-ratios:

$$
\boxed{
\widetilde L_A(e)\ne \widetilde L_A(f),
}
$$

then no potential bridge exists.

Proof.  A potential assigns a single value to
\(\Phi(y^+)-\Phi(y^-)\). \(\square\)

### Corollary 4.3: Triangle Falsifier

For three readout states:

$$
\boxed{
y_0,\ y_1,\ y_2,
}
$$

with edges:

$$
\boxed{
y_0\to y_1,\qquad
y_1\to y_2,\qquad
y_0\to y_2,
}
$$

the bridge requires:

$$
\boxed{
\widetilde L_{01}
+
\widetilde L_{12}
-
\widetilde L_{02}
=0.
}
$$

If this fails, the operational/GCR readout cannot be the source potential for
the GR/SM calibrated ratios.

## 5. Approximate Bridge And Stein Tube

Exact potential agreement is too strong if Route B only controls values up
to a Stein defect.  The approximate bridge is the correct finite target.

### Definition 5.1: Approximate Bridge Defect

Given a candidate potential \(\Phi\), define the edge bridge defect:

$$
\boxed{
\delta_\Phi(e)
:=
\widetilde L_A(e)
-
\bigl(\Phi(y_e^+)-\Phi(y_e^-)\bigr).
}
$$

Let:

$$
\boxed{
\delta_*({\mathcal E})
:=
\inf_\Phi
\max_{e\in{\mathcal E}}|\delta_\Phi(e)|.
}
$$

This is the best possible graph-level mismatch between GR/SM calibration and
operational/GCR potential data.

### Theorem 5.2: Bridge Plus Stein Gives GR/SM Ratio Control

Assume:

1. a potential bridge \(\Phi\) has defect bounded by:

   $$
   \boxed{
   |\delta_\Phi(e)|\le \delta_e;
   }
   $$

2. the Route-B Stein tube on edge \(e\) gives:

   $$
   \boxed{
   |R_e^{act}-R_e^{B}|\le \zeta_e;
   }
   $$

3. the bridge-predicted operational ratio is:

   $$
   \boxed{
   R_e^B
   =
   \exp\{\Phi(y_e^+)-\Phi(y_e^-)\}.
   }
   $$

Then:

$$
\boxed{
|R_e^{act}-R_e^A|
\le
\zeta_e
+
R_e^A\left|1-e^{-\delta_\Phi(e)}\right|.
}
$$

In particular, if:

$$
\boxed{
|\delta_\Phi(e)|\le\delta_e\le1,
}
$$

then:

$$
\boxed{
|R_e^{act}-R_e^A|
\le
\zeta_e
+
2R_e^A\delta_e.
}
$$

Proof.  Since:

$$
\boxed{
R_e^A
=
\exp\{\widetilde L_A(e)\},
}
$$

and:

$$
\boxed{
R_e^B
=
\exp\{\widetilde L_A(e)-\delta_\Phi(e)\}
=
R_e^Ae^{-\delta_\Phi(e)},
}
$$

we have:

$$
\boxed{
|R_e^A-R_e^B|
=
R_e^A|1-e^{-\delta_\Phi(e)}|.
}
$$

The triangle inequality gives the first bound.  For
\(|\delta|\le1\), the elementary estimate
\(|1-e^{-\delta}|\le2|\delta|\) gives the second. \(\square\)

### Consequence 5.3

The bridge has two independent errors:

$$
\boxed{
\begin{array}{ll}
\hbox{bridge defect}:&\delta_\Phi(e),\\
\hbox{dynamical/control defect}:&\zeta_e.
\end{array}
}
$$

Route B recovers Route A only when both are small.

## 12. Exact Bridge Algorithm

The exact bridge is computable by a spanning-tree algorithm.

### Algorithm 12.1: Potential Reconstruction

For each connected component of \(G_Y\):

1. Choose a root vertex \(r\).
2. Set:

   $$
   \boxed{
   \Phi(r)=0.
   }
   $$

3. Choose a spanning tree \(T\).
4. For each tree edge \(u\to v\), define:

   $$
   \boxed{
   \Phi(v)=\Phi(u)+\widetilde L_A(u\to v).
   }
   $$

5. For every non-tree edge \(e:u\to v\), compute:

   $$
   \boxed{
   {\mathsf Res}_T(e)
   :=
   \widetilde L_A(e)-(\Phi(v)-\Phi(u)).
   }
   $$

### Theorem 12.2: Spanning-Tree Residuals Are Complete

The exact bridge exists if and only if:

$$
\boxed{
{\mathsf Res}_T(e)=0
\quad
\hbox{for every non-tree edge }e.
}
$$

Proof.  Each non-tree edge forms one fundamental cycle with the spanning
tree.  The residual is exactly the cycle sum on that fundamental cycle.
Fundamental cycles span the cycle space of the graph, so all cycle sums
vanish if and only if every non-tree residual vanishes. \(\square\)

### Certificate 12.3: Minimal Exact Falsifier

An exact falsifier is one printed non-tree residual:

$$
\boxed{
{\mathsf Res}_T(e)\ne0.
}
$$

This is the most compact way to disprove a potential bridge.

## 13. Approximate Bridge Projection

If exactness fails, we still need to know whether the failure is small,
shrinking, or structurally large.

### Definition 13.1: Least-Squares Bridge Projection

Let \(B\) be the incidence matrix of \(G_Y\).  A potential \(\Phi\) induces
edge differences:

$$
\boxed{
(B\Phi)_e=\Phi(y_e^+)-\Phi(y_e^-).
}
$$

Let \(\ell\) be the vector of corrected log-ratios:

$$
\boxed{
\ell_e=\widetilde L_A(e).
}
$$

The least-squares bridge potential is:

$$
\boxed{
\Phi_*=\arg\min_\Phi \|\ell-B\Phi\|_2^2.
}
$$

The bridge residual is:

$$
\boxed{
r_*:=\ell-B\Phi_*.
}
$$

### Theorem 13.2: Residual Is The Cycle-Space Projection

The least-squares residual \(r_*\) is the orthogonal projection of
\(\ell\) onto the cycle space:

$$
\boxed{
r_*\in\ker B^T,
\qquad
\ell-r_*\in\operatorname{im}B.
}
$$

Thus:

$$
\boxed{
\|r_*\|_2=0
\quad\Longleftrightarrow\quad
\hbox{exact bridge exists}.
}
$$

Proof.  Least-squares normal equations give:

$$
\boxed{
B^T(\ell-B\Phi_*)=0.
}
$$

So \(r_*\in\ker B^T\), the finite cycle space.  The fitted part
\(B\Phi_*\) lies in \(\operatorname{im}B\).  Orthogonal decomposition gives
the claim. \(\square\)

### Corollary 13.3: Approximate Floor Certificate

If:

$$
\boxed{
\|r_*\|_2\ge c>0,
}
$$

for a licensed bridge graph, then no potential bridge can recover the
GR/SM calibrated ratios on that graph with error smaller than \(c\) in
\(L^2\)-edge norm.

This is a real residual floor.
