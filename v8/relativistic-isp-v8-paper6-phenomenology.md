## 1. The seal-decoherence law: the quarter, forced; the equality, dissolved

### 1.1 The quarter law [THEOREM]

For a weak which-path recording (`P₁ = P₀ + sδ`, the record overlap `BC = Σ√(P₀P₁)` multiplying the coherence), the dense-limit Fisher reduction gives `D_B → (1/8)Js²` and `D_KL → (1/2)Js²`, hence `κ = D_B/D_KL → ¼` — distribution-independent (the base *cancels*, sympy-exact); symmetrizing the KL does not help (`κ_symKL = ¼` too, for the averaged symmetrization — the Rényi order, not the convention-dependent value, is the invariant point). Equivalently `−ln BC = σ/4 + (ε²/6)σ + O(σ³)` (the v6 Paper 26 form, citable). The running is quantified exactly on the symmetric binary recorder, with the component closed forms displayed so the ratio is checkable by hand:

> `BC(ε) = √(1 − 4ε²)`,  `D_B(ε) = −½·log(1 − 4ε²)`,  `D_KL(ε) = 4ε·artanh(2ε)`,
> `κ(ε) = −log(1 − 4ε²) / (8ε·artanh 2ε) = ¼ + ε²/6 + 14ε⁴/45 + 724ε⁶/945 + O(ε⁸)`  (sympy-exact),

monotonically increasing from `¼` — strong records decohere *faster* per nat of arrow. The Gaussian recorder (`N(0,1)` vs `N(μ,1)`: `D_B = μ²/8`, `D_KL = μ²/2`) holds `κ = ¼` **at every strength** — an all-order identity, though an elementary one (its ingredients are textbook; Kailath).
