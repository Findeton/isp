The discipline is pre-geometric and single-threaded: every quantity is a record-internal evidence content, character-ledger rank, or index; no continuum field, spinor bundle, or mass scale is used as input — the matter sector's *structure* is read off the records, and the matter sector's *unbuilt* part is named exactly.

## 3. Mass is the record gap when protection breaks [THEOREM + CONJECTURE]

On this base the mass mechanism is literal. The unoriented (vector-like, unprotected) branch is **gapped**, and the gap is a single constant. Receipt `m1` confirms, at `dps = 90`, that

`W = η·θ − log cosh η = 0.1561092001572401165799095`,

where `η = 0.6093778634360062…` is the seal root (`e⁻η = tanh η`, residual `0`) and `θ = tanh η` satisfies the commitment cubic `θ³ + θ² + θ − 1 = 0` (residual `0`). The single number `W` is simultaneously the **spectral gap** `Δ`, the **lightest mass** `m̂(P1)`, and the **work / price of a record event** `W_hist` — all three equal to better than `10⁻⁴⁰`. "Mass = the spectral gap when protection breaks" is therefore not an analogy here: the gap *is* `W`, the same number that prices a record event, recovered when projective protection fails. This is the **theorem** half (v6 paper 7 Thm 9.4): protection broken ⇒ `m̂ = W`, gapped.

The oriented (chiral, protected) branch *breaks* the Griffiths/ferromagnet mechanism that floors the unoriented branch (the local field `h_a = 1.984` exceeds the threshold `η`), and its minimum gap **collapses below `W`** toward gapless. Receipt `m1` gives the descent against `W`:

|---|---|---|
| unoriented (`P1`, the floor) | `0.156109200157` | `= W` |
| oriented `n = 3` (exact) | `0.133530982072` | `< W` |

with successive minimum ratios `m(n)/m(n+1) = 2.069, 2.033, 2.016` — approaching `2` from above — and `m(n)·2ⁿ ≈ 1.0`, slowly decreasing (`1.07 → 1.01` across `n = 3…6`). So the oriented minima fall at least as fast as `2⁻ⁿ` with a prefactor near `1`; the exact asymptotics remain open. The unoriented-gapped / oriented-gapless dichotomy is the PSG dictionary made literal: vector-like = unprotected = gapped at `W`; chiral = protected (carrying the orientation / anomaly structure) = asymptotically massless. But the *gapless* direction is a **conjecture**, not a theorem: the data run through `n = 6` (exact only to `n = 4`; `n = 5, 6` are sampled upper bounds), and whether the oriented minimum reaches `0` at finite `n` or only as `n → ∞` is open (v6 paper 8, the asymptotics question — no prefactor or closed law is claimed). ## 4. The mode-canonicalization wall [NO-GO]

A mass *ratio* requires choosing which mode is which — which ledger is the electron, which the proton. That choice is the matter sector's load-bearing obstruction, and it is a fifth instance of the program's under-determination. Self-consistency fixes the firing law's *form* in every mode: the vector fixed point `∇ψ(h) = e⁻ʰ` holds identically across modes and sectors. What it does not fix is *which* mode is physical, and receipt `m2` (10 of 10 checks, `dps = 100`) shows why no record-internal principle can.

**The structural reason: gauge-inequivalence.** The seal modes differ by **ledger rank** — the number of orthogonal primitive characters, `[1, 3, 7]` along the ladder — an invariant that no record-internal gauge map can change. The per-mode coefficients differ across these sectors (`0.609 / 0.495 / 0.368`, monotone decreasing as coupling dilutes), and no internal map identifies them. The modes therefore sit in **distinct superselection sectors**: genuinely distinct quantum orders, the Wen–PSG hallmark of structure that is canonicalizable only by importing external data. The three non-circular routes that might have selected a canonical mode all fail, in unison, for this one reason:

- **Route 1 — a completeness / no-silent-refinement axiom** (the v6 paper 10 dissolution template). Both horns fail. If the mode label is *recorded*, completeness forces the *maximal* complete ledger, which pushes the coefficient *down* the dilution ladder (`0.6094 → 0.4951 → 0.3680`), **away** from the candidate one-mode value `η_B`, not toward it. If the label is *silent*, the dissolution would apply — but the modes are recorded-distinct (`D(P_{η_B} ‖ P_{h_*}) = 0.00479 > 0`), so that horn does not obtain.
- **Route 2 — an external structural identity.** The tempting bridge `1/η_B = 1.6410179… = ` the 1d Ising correlation length `ξ` is an algebraic **tautology**: `ξ = 1/η` *is* the seal equation `tanh η = e⁻η` rewritten (symbolic residual `0`), so it restates `η_B` rather than selecting it. Worse, the mass-gap length is mode-**covariant**: the coupled mode has its own `ξ(h_*) = 1.2814 ≠ ξ(η_B) = 1.6410`, so "a mode has a mass-gap length" selects nothing, and Paper III already files `1/η_B` as a weight-0 record-*internal* number, not an external selector.
- **Route 3 — a variational principle.** Maximum content selects the *least*-coupled ledger (`η_B`); completeness selects the dilution *limit* — **opposite** selections, neither canonical. The firing clock is mode-blind (`E[I] = 1` in every mode), giving no variational handle, and any in-class extremum is circular because the admissibility gate *is* the mode choice (dropping the count-dual base to `p = 0.2` makes the content `0.526` overshoot `W_* = 0.365` — the gate is load-bearing, the extremum is not free).

**The sole escape relocates the choice.** One route does lie outside these three: a constructed matter Hamiltonian whose energetics select a single ledger rank — Wen's "import external data" move. But that does not close the wall; it *relocates* the free choice to a matter sector that does not yet exist. It changes *where* the choice lives, not *whether* it is free. So mode-canonicalization is the program's **fifth wall**, of the same under-determination shape as the `l_step` scale no-go — self-consistency fixes the form and the ratios, never the last record-blind input — but with a distinct missing input: the canonical *mode*, not an absolute scale. The records are structurally barred from a canonical mode, and the wall is the result.

## 5. The hierarchy mechanism is form-only [FORM-ONLY]

The gravitational coupling per species, `c_m = G m² / ℏc ~ 10⁻³⁹`, does get a genuine *mechanism* from the records — but only in form. The record(slow) / substrate(fast) split makes the smallness a **mode-separation ratio**: a `1/N` suppression by a large mode / species / event count, structurally a near-isomorph of three established mechanisms (Sakharov induced gravity, the Dvali species scale, the matrix slow/fast split), all weight-0 and all running on SHARD's two-mode record / substrate structure. So the *shape* of the hierarchy — why `c_m` is a tiny pure number — is mechanistic, not arbitrary.

But the controlling count `N = 1/c_m ~ 10³⁸` is itself weight-0 and **not corpus-forced**: the natural record gaps are `O(1)` to `O(10)`, some `37` orders short of the required count. The mechanism fixes that `c_m` *is* a `1/N` ratio without fixing `N`, so exactly one free per-species calibration survives — which is precisely Paper V's result `c_m = γ_G·ν_m²` (the gravitational coupling per species is a free calibration). The hierarchy mechanism is therefore **form-only**: a derived structure for the smallness, with the magnitude left to the one calibration the records cannot supply.

## 6. No mass ratio is computable now [GATED]

The two parts compose to a clean negative on the headline target. Computing even one physical mass ratio as a record-gap pure number is **doubly gated**. First, by the mode wall (§4): the three observables a ratio would draw on — the spacing `d`, the coupling `c_m`, the per-mode masses `ν_m` — are all firing-fixed-point outputs, and the fixed point is rank-degenerate (`0.6094 / 0.4951 / 0.3680`, no canonical member), so "this mode is the electron, that mode is the proton" is record-internally undecidable. Second, by two unbuilt prerequisites: the record's ledger-chirality (orientation classes, an Ising-level structure) is not yet identified with spinor / PSG chirality, and the protected ⇒ gapless asymptotics is conjectural (§3). SHARD computes mode-*relative* gaps, the chiral `2⁻ⁿ` collapse ratios, and the hierarchy *form* — but not the absolute identification a mass ratio needs. **No mass prediction is licensed by the executed corpus**, and the paper makes none.
